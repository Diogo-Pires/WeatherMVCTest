# WeatherMVCTest
A .NET MVC application designed to fetch and display weather data using the OpenWeatherMap API.​

**This was a test from an interview process that needed to be done in less than one day.**

## Overview
WeatherMVCTest is a web application built with ASP.NET MVC that allows users to retrieve and view current weather information for specified locations. The application demonstrates the integration of external APIs within the MVC architectural pattern.​

## Features
 - User Input: Enter a city name to retrieve current weather data.
 - API Integration: Fetches real-time weather information from the OpenWeatherMap API.
 - MVC Architecture: Utilizes the Model-View-Controller pattern for organized code structure.
 - Responsive Design: Ensures usability across various devices and screen sizes.​

## Technologies Used
 - ASP.NET MVC
 - C#
 - HTML/CSS
 - JavaScript
 - OpenWeatherMap API​

## Getting Started
### Prerequisites
 - Visual Studio 2022 or later
 - .NET 6 SDK or later
 - An OpenWeatherMap API key​

### Installation
1. Clone the repository:​
GitHub
```bash
git clone https://github.com/Diogo-Pires/WeatherMVCTest.git
```
1. Open the solution file WeatherMVCTest.sln in Visual Studio.​

1. Restore NuGet packages and build the solution.​

1. Configure your OpenWeatherMap API key in the application's configuration file (e.g., appsettings.json or directly in the code, depending on implementation).​

### Running the Application
 - Press F5 or click on the "Start" button in Visual Studio to run the application.

 - Navigate to the application's URL in your browser (typically https://localhost:5001/).

 - Enter a city name in the provided input field to retrieve and view the current weather information.​
